## Quick Start

Get up and running with pyrig in seconds:

<CodeGroup>

```bash uv (recommended)
uv init
uv add pyrig
uv run pyrig init
```

```bash pip
pip install pyrig
pyrig init
```

</CodeGroup>

That's it! pyrig generates your complete project structure, tests, CI/CD, and documentation.
